# Pick2

Real-time multiplayer card game using Node.js, Express, and WebSockets.

---

## Quick Start

```bash
npm install
npm start
```

Open:
http://localhost:8080

---

## Tech Stack

- Node.js
- Express
- ws
- Jest

---

## Features

- Real-time multiplayer rooms
- AI players
- Turn-based gameplay
- Server-authoritative state
- Inactivity cleanup

---

## Environment

```env
PORT=8080
```

---

## Scripts

```bash
npm start

```
